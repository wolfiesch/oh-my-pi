# @oh-my-pi/appserver

This package is now a thin compatibility launcher. It re-exports the checksum-pinned
`@t4-code/host-service`; OMP's coding-agent package supplies only the authority that understands
OMP sessions, locks, workers, settings, models, credentials, and native events.

Generic WebSocket transport, remote policy, security, projections, indexes, workspace handling,
and transcript paging are implemented and tested in T4.
